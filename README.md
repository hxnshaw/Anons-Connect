# Anons-Connect

# INTRODUCTION
Anons-Connect is an anonymous blogging website where users signup and connect with strangers on the internet. When users signup, they are known as Anon(s).
They do not need to share their personal information.
Users come up with a username, and are known by it. Informations such as email, date of birth, residence are not shown to the public. 

# FEATURES
* User registration, authentication, login and logout
* Only registered anons can view and create posts
* Anons can create posts, edit and delete their posts.
* Anons can follow and unfollow fellow anons.
* An anon can like a post from another anon.
* An anon can also make comments on the posts of other users.
* Anons have a timeline where they can see their posts as well as posts from people they follow.

# ADMINS
# Admins are super-users, and have alot of priviledges, and they include:
* View all users
* View all posts and comments from all registered users
* Access to all routes
* Ability to delete an anons post or comments if it breaks set rules.
* Admins can also delete user profiles.
